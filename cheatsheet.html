HTML5 Cheat Sheet

Essential tags for web structure and content.

Tag Description Example
<!DOCTYPE html> Defines the document type (not an HTML tag). <!DOCTYPE html>
<html> The root element of an HTML page. <html lang="en">
<head> Contains metadata (title, scripts, styles). <head>...</head>
<body> Contains all visible page content. <body>...</body>
<h1>-<h6> Defines headings. H1 is most important. <h2>Section Title</h2>
<p> Defines a paragraph of text. <p>Paragraph content.</p>
<div> A generic block container for grouping content. <div class="container">
Tag Description Example
<a> Anchor tag, used to create hyperlinks. <a href="url">Link Text</a>
<img> Embeds an image into the document. <img src="pic.jpg" alt="Desc">
<ul> Unordered list (bullets). <ul><li>Item</li></ul>
<ol> Ordered list (numbered). <ol><li>Item</li></ol>
<table> Defines a standard HTML table. <table><tr><td>...</table>
<link> Links external resources (like CSS). <link rel="stylesheet" href="style.css">
<script> Embeds or links JavaScript. <script src="script.js"></script>
Tag Description Example
<form> Defines an HTML form for user input. <form action="url" method="POST">
<input> An input field for user data. <input type="email" name="user_email">
<label> Defines a label for an <input> element. <label for="name">Name</label>
<button> A clickable button. <button type="submit">Send</button>
<textarea> A multi-line text input control. <textarea rows="5"></textarea>
